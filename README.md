# 📧 Automated Payroll PDF Generator & Bulk Email Sender

## 🚀 Project Overview

**Automated Payroll PDF Generator & Bulk Email Sender** is an Excel VBA-based automation solution designed to streamline the payroll distribution process. The system automatically generates individual salary slip PDFs for employees and sends them directly to their registered email addresses with a single click.

This project eliminates repetitive manual work, reduces processing time, and ensures accurate payroll communication across the organization.

---

## 🎯 Key Features

### ✅ Automatic PDF Generation

* Generates employee-wise salary slip PDFs directly from Excel data.
* Creates multiple PDF files in a single execution.
* Dynamic file naming based on Employee Name, Employee ID, Month, or Pay Period.

### ✅ Bulk Email Automation

* Sends salary slips to hundreds of employees simultaneously.
* Automatically fetches employee email addresses from Excel.
* Personalized email subject and message for each employee.
* Attaches the corresponding PDF salary slip automatically.

### ✅ Excel VBA Automation

* Fully developed using Microsoft Excel VBA.
* User-friendly interface with minimal manual intervention.
* One-click execution process.

### ✅ Error Handling & Validation

* Validates email addresses before sending.
* Tracks successful and failed email deliveries.
* Prevents duplicate processing.

### ✅ Time-Saving Solution

* Reduces manual payroll distribution effort by over 90%.
* Suitable for HR, Payroll, Finance, and Administrative departments.

## 📂 Project Workflow

1. Import employee payroll data into Excel.
2. Enter employee email addresses.
3. Click the automation button.
4. Generate individual salary slip PDFs.
5. Attach generated PDFs automatically.
6. Send personalized emails to all employees.
7. Generate processing log/report.
